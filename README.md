# MathGenius 🧮

**AI-powered math tutor for complex problems with step-by-step explanations**

✅ **100% FREE** - Uses local Ollama AI  
✅ **No API Keys** - No subscriptions  
✅ **Open Source** - Run anywhere  
✅ **Step-by-Step** - Learn how to solve problems  

---

## Quick Start (5 minutes)

### 1️⃣ Install Ollama
- **Mac/Windows**: Download from https://ollama.ai
- **Linux**: `curl https://ollama.ai/install.sh | sh`

### 2️⃣ Download Free AI Model
```bash
ollama pull mistral
```
(Keep `ollama serve` running in background)

### 3️⃣ Run MathGenius
```bash
git clone https://github.com/Aetherion108/MathGenius.git
cd MathGenius
pip install -r requirements.txt
python main.py
```

### 4️⃣ Ask a Problem
```
📌 Problem: Solve x² - 5x + 6 = 0
```

Done! 🎉
